# Начало работы

## Шаг 1: Установка приложения

### Для Android
1. Откройте RuStore
2. Найдите "SketchShare"
3. Нажмите "Установить"
4. Дождитесь завершения установки

### Для iOS
1. Откройте App Store
2. Найдите "SketchShare"
3. Нажмите "Загрузить"
4. Введите пароль при необходимости

## Шаг 2: Регистрация

1. Откройте приложение
2. Нажмите "Зарегистрироваться"
3. Заполните поля:
   - Имя пользователя
   - Email адрес
   - Пароль
4. Подтвердите email

## Шаг 3: Создание первого скетча

### Инструменты рисования
- **Кисть** (`B`) - для широких линий
- **Карандаш** (`P`) - для тонких линий
- **Ластик** (`E`) - для удаления
- **Заливка** (`F`) - для закрашивания областей

### Сочетания клавиш
| Действие | Android | iOS |
|----------|---------|-----|
| Отменить | Ctrl+Z | Cmd+Z |
| Повторить | Ctrl+Y | Cmd+Y |
| Сохранить | Ctrl+S | Cmd+S |
| Новый файл | Ctrl+N | Cmd+N |

**Совет:** Используйте жесты двумя пальцами для масштабирования холста.
