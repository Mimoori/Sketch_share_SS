# Функции приложения

## Основные функции

### 1. Система рисования
- Поддержка слоев
- Различные кисти и текстуры
- Настраиваемая палитра цветов
- Поддержка давления (для стилусов)

### 2. Управление проектами
- Создание папок
- Тегирование скетчей
- Поиск по названию и тегам
- Сортировка по дате создания

### 3. Экспорт и импорт
- Форматы: PNG, JPEG, SVG
- Настройка качества
- Пакетный экспорт
- Импорт изображений
